Data models

The following guides describe common tasks related to modeling data in your application.

Components

Using composition
Create a parent-child (has-a) hierarchy of components using property decorators.

Restricting component bindings
Use advanced component presets to limit which properties can be bound.

Handling component events
Use Typescene’s event mechanism to communicate state changes and user actions between components.

Using observers
Observe a component class to handle events and property changes on all instances.

Data structures

Using records
Use managed record classes to keep track of structured data.

Using lists and maps
Keep track of collections of objects using Typescene’s managed list and map types.

String formatting

Formatting strings
Use the strf method to format text using a format string and any number of parameters.

Using plural forms in formatted strings
Include plural forms in format strings to change word forms or endings based on a numeric parameter.
