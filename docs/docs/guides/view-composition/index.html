View composition

The following guides describe common tasks related to composite views.

Lists

Displaying lists of items
Use the list controller view to display a component for each item in a list.

Using events with lists
Handle list cell events to allow users to perform actions on list items.

Paginating lists
Use list controller properties to paginate items within a list.

Nesting lists
Use nested list views to display component hierarchies.

Managing list item selection
Use selection events to manage selection states for components within a list view.

Scroll containers

Displaying scrolling content
Use scroll containers to allow users to scroll content up and down within a container.

Using scroll events
Handle scroll events to respond to updates in scroll position within a container.

Snapping to scrolling list items
Use scroll snapping to snap to components within a scroll container.

Other views

Displaying views conditionally
Display or remove content using conditional view controllers.

Managing component selection
Use selection events to limit component selection to a single item within a selection view controller.

Nesting other views
Use the view renderer controller to nest views from outside of the view hierarchy.
