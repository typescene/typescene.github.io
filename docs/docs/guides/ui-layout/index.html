UI layout

The following guides describe common tasks related to laying out UI components.

Rows and columns

Arranging components in rows
Use rows to align UI components horizontally.

Spacing components in rows
Components can be aligned and resized within rows.

Arranging components in columns
Use columns to align UI components vertically.

Cells

Arranging components in cells
Use cells to divide and decorate screen space.

Animating cells
Reveal and animate content using cell transitions and animations.

Separators

Displaying separators
Divide your content visually using separator components.

Adding container spacing and separators
Divide your content visually using separators and spacing within containers.

Styles for UI layout

Changing component dimensions
Set specific styles to determine the size of any UI component on screen.

Positioning components
Set specific styles to control the position of any UI element on screen.

Customizing component arrangements
Set container styles to change how UI components and other containers are arranged within containers.
