ManagedListChangeEvent

class ManagedListChangeEvent

extends ManagedChangeEvent

Base type for events that are emitted when changes occur to a ManagedList or ManagedMap.

Note: This event is emitted by ManagedList and ManagedMap and should not be emitted by your application.
